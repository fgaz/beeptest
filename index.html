Beep Test

Audio per il Test della Navetta di Léger (o Beep Test)

Visualizza il progetto su GitHub

Cos'è

Il test della navetta è un test di resistenza in cui l'atleta deve correre fra due coni distanti 20 metri l’uno dall’altro secondo una cadenza sincronizzata.

Scarica i file audio

Sono disponibili a questo indirizzo vari file audio pregenerati e già pronti per l'utilizzo.
